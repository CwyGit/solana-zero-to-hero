## 🎯 课程回顾

在本课程中，你掌握了以下重要知识：

### 第1章：NFT基础

✅ **NFT的核心属性**
- 不可分割性 - 不能拥有0.5个NFT
- 独特性 - 供应量为1
- 固定供应量 - 无铸造权限
- 可分组性 - 可组织成集合
- 丰富的元数据 - 名称、描述、图像等

**比喻：** 理解了NFT的"身份证"！

---

### 第2章：Token Metadata

✅ **Metaplex Token Metadata程序的账户结构**
- `Metadata`账户 - 描述层
- `Master Edition`账户 - 权限管理
- `Token Record`账户 - pNFT状态

**比喻：** 学会了传统NFT方案！

---

### 第3章：Metaplex Core

✅ **Metaplex Core程序的创新设计**
- 单账户设计 - 提高效率
- 插件系统 - 灵活扩展
- 降低成本 - 大幅减少费用

**比喻：** 掌握了新一代NFT方案！

---

## 📊 Token Metadata vs Core

| 特性 | Token Metadata | Core |
|------|---------------|------|
| 账户数量 | 多个（复杂） | 单个（简单） |
| 依赖 | SPL-Token | 独立 |
| 成本 | 较高 | 大幅降低 |
| 灵活性 | pNFT | 插件系统 |
| 推荐 | 老项目 | 新项目 |

**小伙伴们要特别注意啦：**

> 💡 新项目建议使用**Metaplex Core**，它更简单、更便宜、更灵活！
