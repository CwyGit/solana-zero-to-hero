## 🔑 Secp256r1签名 (课程15)

**用途**：支持WebAuthn/Passkey登录

**特点**：
- 与浏览器WebAuthn标准兼容
- 支持硬件安全密钥
- 无需私钥管理

```rust
// 需要调用Secp256r1预编译程序
use solana_secp256r1_program;
```

---

## 🔑 Winternitz签名 (课程16)

**用途**：抗量子计算的签名方案

**特点**：
- 基于哈希的签名
- 理论上抗量子攻击
- 签名体积较大

---

## 🔑 Assembly汇编 (课程17)

**用途**：极致性能优化

**使用场景**：
- 关键路径优化
- 减少CU消耗
- 程序大小优化

```rust
// 内联汇编示例
unsafe {
    asm!("add {0}, {1}", inout(reg) result, in(reg) value);
}
```

---

## 🔑 指令内省 (课程18)

**用途**：在程序中检查当前交易的其他指令

**应用场景**：
- 验证交易包含特定指令
- 闪电贷保护
- 多指令原子操作验证

```rust
use solana_program::sysvar::instructions;

// 获取当前交易的所有指令
let ixs = instructions::load_instruction_at_checked(index, account)?;
```
