# 03-销毁代币 - Burn烧掉！🔥

嘿，小伙伴！👋

**销毁(Burn)**让你永久删除代币，减少总供应量！

**比喻说明：** 就像撕掉钞票，让钱永远消失！

---

## 🎯 什么是销毁？

**功能：** 永久删除代币

**用途：**
- 🔥 通缩机制
- ♻️ NFT销毁
- 📉 减少供应

---

## 💡 使用方法

```typescript
import { burn } from '@solana/spl-token';

await burn(
    connection,
    payer,
    account, // 代币账户
    mint, // mint地址
    owner, // 所有者
    amount // 销毁数量
);
```

---

## ❓ 常见问题

### Q: 销毁后能恢复吗？

**答：** 不能！永久删除！
