# 04-关闭账户 - 回收租金！💰

嘿，小伙伴！👋

**关闭账户**让你删除空的代币账户并回收租金！

**比喻说明：** 就像注销银行卡退回押金！

---

## 🎯 什么是关闭账户？

**功能：** 删除余额为0的账户

**好处：**
- 💸 回收SOL租金
- 🧹 清理账户
- 📊 优化管理

---

## 💡 使用方法

```typescript
import { closeAccount } from '@solana/spl-token';

await closeAccount(
    connection,
    payer,
    account, // 要关闭的账户
    destination, // 接收SOL的账户
    owner // 所有者
);
```

---

## ⚠️ 注意

**条件：** 余额必须为0！

---

**现在小伙伴们懂了吧？** 及时关闭不用的账户！💰
